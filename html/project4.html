Location Logger

Overview

Location Logger is a web application that allows the user to journal their past trips. The user may click anywhere on the map and they are introduced with a form that captures a description, photo, and date of visitation. The input is saved into the database and the location is saved and displayed using a popup marker.

Process

Location Logger was created using React as the framework, MongoDB for the database, Express for the routing, and MapBox for the rendering of the map. The dependencies used throughout the process of development and production are nodemon, eslint, mongoose, helmet, and morgan.
