Image Classifier

Overview

The ocean is filled with thousands of different creatures and it is extremely difficult to remember what species they are and what they look like. This application allows you to upload an image to identify an unknown species. Process

I used React as the framework of the application and TensorFlow to identify the species.

The application consists of four states. Each state is triggered by an action that transitions to the next state.

State 1.
The initial button triggers the TensorFlow model to load and saves it to the component state.

State 2.
Once the model is loaded, the user can upload an image and it is converted to an object URL. The object URL is used as the src property of an IMG file to display.

State 3.
The image is passed to the model to classify the species and store the result into state.

State 4.
The final state displays the result and clears the image URL. Then transitions back to the image upload, since the model is already loaded into state.
