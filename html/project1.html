Playlist Automation

Overview

Have you ever liked a song on Youtube and then added it to your Spotify Playlist? This application simplifies the process by automatically transferring your Youtube playlist items to your desired Spotify playlist.

Process

The playlist automation uses React as the framework to develop the backbone of the application and utilizes both the Youtube API and the Spotify API.

This project can be broken down into three parts:

Part 1.
Use the fetch API to GET the video title from Youtube, then parse the JSON response so that the song title and artist name saves into memory.

Part 2.
The song title and the artist name is stored to query the Spotify API and retrieve the related track. Then the track's id is extracted from the returned JSON.

Part 3.
Once the track ID is saved into memory it is used to POST the track in the desired Spotify playlist.
