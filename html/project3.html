Sorting Algorithm Visualizer

Overview and Process

This project's purpose is to visualize the process of sorting algorithms. The algorithms displayed are bubble sort, heap sort, insertion sort, merge sort, quick sort, and selection sort. The application was developed using JavaScript, HTML, and CSS.

An array stores div tags that are randomly sized according to the user's desired length, and the user has the option of choosing which algorithm to visualize. Once an algorithm button is clicked, the function corresponding to the button triggers.

The workflow of the project prioritizes the separation of concerns regarding each sorting algorithm and its function.
